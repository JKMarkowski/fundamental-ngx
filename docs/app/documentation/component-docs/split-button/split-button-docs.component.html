Button Options

There are three emphasis styles used to indicate the importance of the button on the page. Do not provide a value for the default option. Use [options]="'emphasized'" or [options]="'light'" to set the option.

Button Types

The default button type is "action", no modifier is required. Use [fdType]="'standard'", [fdType]="'positive'", [fdType]="'medium'", or [fdType]="'negative'" to modify the type of the button.

Buttons with Icons

Use [glyph]="'{{ '{' }}icon-name}'" to change an icon of dropdown menu.

Open/Close programmatically

There is possibility to control open/close of popover by 2-ways data binding [(isOpen)]

Adding template to action button

There is possibility to add some html template to action button with fd-split-button-action-title directive
